## Getting Started

I'm learning haskell doing mini tests on [HackerRank](https://www.hackerrank.com/).
You should try too!

### Prerequisites

- `ghg` to run haskell programs

## Usage

Use ghc to make a executable file of any haskell program

```sh
ghc -dynamic FILE.hs
```

And execute that program with

```sh
./FILE
```
